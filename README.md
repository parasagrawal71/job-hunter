# Job Hunter CLI

> unzip job-hunter.zip   
> cd job-hunter 
<br />   

> python -m venv venv   
> source venv/bin/activate 
<br />   

> pip install -r requirements.txt   
> playwright install   
<br />   

> pip install -e . 
<br />   

> Run:    
[Testing]   
> job-hunter --input companies_test.csv   
> Open frontend/index.html for dashboard   
<br />   

[All companies list]   
> job-hunter --input companies.csv
